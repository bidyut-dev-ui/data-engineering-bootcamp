# Generative AI/ML Engineer

## About the Role
We are looking for a talented Generative AI/ML Engineer with 2–4 years of experience in Python and backend development. In this role, you will work with modern AI frameworks, vector databases, and backend technologies to build scalable, intelligent systems. If you enjoy solving complex problems and want to shape real-world AI applications, we’d love to hear from you.

## Key Responsibilities
- **[#skill-ai-rag]** AI/ML Development & Optimization: Build and integrate Generative AI and ML models using LangChain, Llama-Index, Scikit-learn, TensorFlow, or PyTorch, and optimize them through hyperparameter tuning, cross-validation, and observability tools for accuracy and reliability.
- **[#skill-ai-rag]** RAG, Embeddings & Vector Search: Implement prompt engineering, embeddings-based similarity search, and RAG workflows using vector databases such as Pinecone, Weaviate, Supabase, or PGVector.
- **[#skill-ai-rag]** Multimodal Data Processing: Work with text, image, and video datasets, contributing to preprocessing, feature engineering, and lightweight multimodal data pipelines.
- **[#skill-fastapi]** Backend & API Engineering: Build scalable backend services using FastAPI, Django, or Flask, design clean RESTful APIs, and integrate third-party SDKs, AI APIs, and external systems.
- **[#skill-database]** Database & Data Workflow Management: Work with PostgreSQL, MySQL, and MongoDB, and integrate vector search capabilities to support AI-driven workflows and backend pipelines.
- **[#skill-python]** Task Automation & Scheduling: Manage background tasks and scheduled jobs using Celery, cron jobs, or similar queueing systems.
- **[#skill-docker]** Containerization & Deployment: Use Docker for application packaging and ensure reliable, consistent deployments across environments.
- End-to-End Development: Contribute to the complete software development lifecycle from requirement analysis and design to development, testing, and deployment.

## Required Qualifications
- Education & Experience: Bachelor’s degree in Computer Science, Artificial Intelligence, Data Science, Engineering, or a related field, with 2–4 years of hands-on experience in Python development, backend engineering, and AI/ML workflows.
- **[#skill-python]** **[#skill-fastapi]** Python & Backend Engineering: Strong proficiency in Python with experience using FastAPI, Django, or Flask, along with solid understanding of OOP, clean code principles, and API development using RESTful services and third-party integrations.
- **[#skill-ai-rag]** AI/ML & RAG Expertise: Working knowledge of Generative AI frameworks (LangChain, Llama-Index), ML libraries (Scikit-learn, TensorFlow, PyTorch), and hands-on experience with embeddings, similarity search, and vector databases like Pinecone, PGVector, Weaviate, or Supabase.
- **[#skill-database]** Database & Data Handling: Familiarity with relational and non-relational databases (PostgreSQL, MySQL, MongoDB), along with experience in preprocessing and handling text, image, and video data for AI/ML pipelines.
- **[#skill-devops]** Testing, Monitoring & Optimization: Proficiency with Pytest or similar testing frameworks, and exposure to observability tools for debugging, monitoring, and performance optimization of AI/ML systems.

## Preferred Skills
- **[#skill-devops]** Cloud Platforms: Experience with AWS, Azure, or GCP for deploying AI/ML workloads.
- AI/ML Ecosystem: Familiarity with Hugging Face, OpenAI APIs, or model-serving tools like TensorRT, ONNX Runtime, or FastAPI-based inference servers.
- **[#skill-docker]** Containerization & Orchestration: Experience with Docker, Kubernetes, or serverless platforms for scalable deployments.
- **[#skill-devops]** DevOps & CI/CD: Exposure to automated pipelines, GitHub Actions, Azure DevOps, or similar DevOps environments.
- Security: Understanding of secure API development, data protection, and best practices for handling ML/AI systems in production.
